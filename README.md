U2 - Actividad#1 - Brandon CB

El programa consiste en pedir una cantidad de personas que desea leer edad y nombre, dependiendo la cantidad de personas será lo que hará el programa. 
Si ingresa 1 persona, se le pedirá nombre y edad, posteriormente se dirá si es mayor o menor de edad. 
Si ingresa más de 1 personas, se pedirán nombres y edades, y se guardará en un arreglo, después se verificará 1x1 si es mayor o menor y se irán imprimiendo por listas separadas.
Se puso un poco de seguridad para que el usuario no pueda ingresar datos erroneos y el programa pueda correr sin ningún error.

Lenguaje: C#

Autor: BCB
